# Excel Markdown Reformatter

Excel方眼紙ファイルをMarkdownに変換し、読みやすい形式に再フォーマットするClaude Codeプラグイン

## Installation

```bash
/plugin marketplace add https://github.com/nyankotaro/claude-excel-markdown-reformatter
```

### 前提条件

markitdownライブラリが必要です。

```bash
pip install markitdown
```

## Plugins

### excel-md

markitdownで生成されたExcelマークダウンファイルを読みやすい形式に再フォーマットするプラグイン

詳細は [plugins/excel-md/README.md](plugins/excel-md/README.md) を参照してください。
